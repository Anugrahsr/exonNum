# ExonNum 
                                                             
### Exon Number Finder v.1 
A tool to find genes of user specific exon number

## Installation
git clone https://github.com/CyPH3R-ASR/exonNum.git

## Usage
1. cd exonNum
2. chmod 775 exonNum.sh
3. ./exonNum.sh
4. Enter number of Exon
